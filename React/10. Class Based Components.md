# Class Based Components :

# 1.  Syntax :

```js
import React from "react";

function App() {
    return (
        <div>App</div>
  );
}
```


```js
class App extends React.Component {
  render() {
    return <div>React Class Based Component - App</div>
  }
}

export default App;
```
