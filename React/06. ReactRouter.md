# 2. Replacement of anchor tag by Link in React Router:

`FileName: Navbar.js` ‚¨áÔ∏è
```js
// abb: rfc
import React from 'react'

// react router dom ------------------
import { Link } from "react-router-dom";    // ‚≠ê‚≠ê‚≠ê

export default function Navbar() {
    return (
        <>
            <div className='navbar'>
                <p>Anchor Tag : </p>
                <a href="/" >Home</a>
                <a href="/about" >About</a>
                <a href="/contact" >Contact</a>                
            </div>
            
            <div className='navbar'>
                <p>Link of React Router : </p>
                <Link to="/" >Home</Link>           // ‚≠ê‚≠ê‚≠ê
                <Link to="/about" >About</Link>     // ‚≠ê‚≠ê‚≠ê
                <Link to="/contact" >Contact</Link> // ‚≠ê‚≠ê‚≠ê          
            </div>
        </>
    )
}
```
  - `a` is replaced by `Link`
  - `href` is replaced by `to`
